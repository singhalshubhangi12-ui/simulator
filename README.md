Traffic Light Simulator

A simple Traffic Light Simulator built using HTML, CSS, and JavaScript.

Features

* Automatic light switching
* Red → Yellow → Green cycle
* Start and Stop buttons
* Dynamic UI updates using JavaScript

Technologies Used

* HTML
* CSS
* JavaScript

How It Works

The simulator changes traffic lights automatically every few seconds.

Flow:

🔴 Red → 🟡 Yellow → 🟢 Green → 🔴 Red

JavaScript keeps track of the current light using a state variable.

The project uses:

setInterval()

to repeatedly update the traffic signal.
